# MyTab Portal

一个 iTab / 今日热榜风格的新标签页扩展（Chrome Manifest V3），支持简洁首页 + 热榜分栏、数据源配置和本地缓存秒开。

## 主要能力

- 双页布局
  - 左侧菜单（首页 / 热榜）
  - 首页保持简洁（时钟、快捷入口、待办、倒数日）
  - 热榜页集中展示聚合卡片墙
  - V2EX
  - 虎嗅
  - 36Kr
  - IT之家
  - 澎湃
  - 财联社
- 热点抓取与质量优化
  - 每个站点优先抓取频道页/列表页，再回退首页
  - 源级路径规则，过滤站点首页/非正文链接
  - 清洗 markdown 标题里的图片和 # 前缀，减少脏数据
  - 每个站点展示最新 20 条（抓不到则显示“暂无可用内容”）
- 源配置
  - 启用/禁用开关
  - 拖拽排序（从上到下）
  - 设置持久化到 `chrome.storage.local`
- 编辑模式
  - 倒数日输入区仅在编辑模式显示
  - 待办输入与勾选/删除控件仅在编辑模式显示
  - 快捷图标卡支持自定义添加和删除，仅在编辑模式可操作
- 快捷图标卡
  - 支持用户自定义网址
  - 支持重置默认

## 抓取策略

通过 `r.jina.ai/http://目标站点` 抓取文本后解析链接，优先提取站内内容页，过滤登录/关于等非内容链接。

## 本地调试

1. 打开 `chrome://extensions`
2. 开启开发者模式
3. 加载已解压扩展，选择本仓库目录

## CI 打包

工作流：`Build Browser Extension`

- `workflow_dispatch` 手动触发
- 推送到 `main` 或提交 PR 自动触发

产物：`mytab-lite.zip`（压缩包根目录直接是扩展文件，不额外套目录）


## 性能说明

- 首屏优先读取 `chrome.storage.local` 的热榜缓存并立即渲染。
- 后台并发刷新全部启用源，并设置单请求超时，避免首次打开长时间空白等待。
